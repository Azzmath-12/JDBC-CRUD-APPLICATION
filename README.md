# JDBC CRUD Application

A simple Java console-based application that demonstrates **CRUD (Create, Read, Update, Delete)** operations using **JDBC** with a **MySQL** database.

This project is built for learning and practicing core JDBC concepts such as `Connection`, `Statement`, `PreparedStatement`, and `ResultSet`.
